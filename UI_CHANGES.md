## User Experience Flow

1. **User opens Add Event screen**
   - Sees contact number field with South Africa as default
   - Flag and country code visible

2. **User clicks country selector**
   - Modal opens with searchable country list
   - Sees 200+ countries with flags

3. **User searches for country**
   - Types country name
   - List filters in real-time

4. **User selects country**
   - Modal closes
   - Selected country flag and code appear
   - Focus moves to phone number input

5. **User enters phone number**
   - Types number without country code
   - Sees automatic formatting
   - Gets instant validation

6. **User saves event**
   - Complete international number stored
   - Format: +[country code][number]
